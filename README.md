# Vyride

This repository contains the Vyride web app scaffolded from a template.

## Local development

Requirements: Node.js and npm.

Install and run:

```bash
npm install
npm run dev
```

## Deploying

Build for production:

```bash
npm run build
```

Serve the `dist/` folder or deploy to your preferred hosting provider.

This project originally included generator-specific metadata which has been removed.
